Terminal-Bench 2.0

Terminal-Bench is a benchmark for evaluating AI agents and language models on realistic command-line tasks inside containerized environments.

Terminal-Bench 2.0 aims to push beyond current frontier benchmarks by introducing harder, more realistic, and higher-value tasks that better measure real-world agent capabilities.

Overview

Terminal-Bench evaluates whether an agent can successfully:

Navigate complex codebases
Debug software systems
Use Linux and CLI tooling effectively
Work with large contexts and documentation
Perform multi-step reasoning
Complete realistic engineering workflows

Tasks run in isolated containerized environments and are designed to reflect practical work performed by software engineers, researchers, and technical operators.
